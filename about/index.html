About Antheiz Radio
Home

Antheiz Radio is a personal project born out of a simple need: to have a single, easy-to-use platform for my favorite radio stations. It's not a commercial product, but rather a solution I created for myself to enjoy the radio sources I love, all in one place.

The main goal of this project was to create a minimalist, user-friendly interface that allows me to listen to my preferred radio stations effortlessly. By sharing it as an open-source project, I hope it might also be useful for others who have similar needs.

Antheiz Radio is created by Antheiz and is open-source. If you're interested in how it's built or want to adapt it for your own use, you can find the source code on GitHub at https://github.com/antheiz/radio.

While this project was created for personal use, I welcome anyone who finds it helpful to use and contribute to it. Enjoy your listening experience!
